Основные виды селекторов:

1.Селектор по тегу(элементу):

Выбирает элемент по имени тега. Пример:

p { padding-bottom: 15px; }

Селектор выберет все элементы с тегом "p" на странице

2.Селектор по классу:

Выбирает элемент по имени класса. Пример:

.center {text-align: center;}

селектор .center выберет все элементы, имеющие класс center

3.Селектор по идентификатору (id):

Селектор по идентификатору предназначен для выбора элемента по идентификатору (значению атрибута id). Пример:

#footer { height: 50px; margin-top: 20px; }

селектор #footer выберет элемент, имеющий идентификатор footer

4.Мультиклассы:

.class1.class2 { Описание правил стиля } К любому элементу одновременно можно добавить несколько классов, перечисляя их в атрибуте class через пробел. Пример:

a href="#" class="tag level6">Paint.NET

к элементу а применяться оба класса tag и level6

5.Вложенные селекторы:

E F { Описание правил стиля }. Например:

ul li

Стили будут применены к дочерним элементам li, которые вложены в элемент ul.

Другие селекторы:

  • * - универсальный селектор. Выбирает все элементы страницы
  • A>B - Дочерний селектор. Выбирает только элемент, кот. располагается внутри родительского элемента
